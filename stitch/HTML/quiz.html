If W(t) is a standard Wiener process, what is the expected value of E[W(t)²] at time t?

auto_awesome

Brilliant! Correct Answer

+50 XP

By definition of the Wiener process, the variance Var(W(t)) = E[W(t)²] - (E[W(t)])². Since E[W(t)] = 0, it follows that E[W(t)²] = Var(W(t)) = t.

lightbulb Quick Tip Remember: W(t) ~ N(0, t)
